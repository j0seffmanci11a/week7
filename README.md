# week7
Week 7 Exercise: Mid-Term Project Kickoff for DIG4503c

For this midterm project, I want to make an app that is almost like a mix of a list and a to-do app, which I like because they help keep me in order. 
I wish there were a web-based app that was a list first and could tell you how long it took to complete an assignment from the date of creation to the date you finished it, which is where the date-tracking part of the app comes in, and view it in either a list format or a calendar. 
What I also wish some to-do and list apps would have is to make little notes about that assignment, or if you want the list to be simple, but feature rich, for the header of the assignment or task, then you could go into a little bit more detail but not have it be 3-5 sentences like a simple description of a movie or tv show on a guide. 
I think having a timer feature could help you complete that task, and you can not only see how long you had that task for, but you can also see how long it took you when you decided to work on it in that focus window. 
Another thing I think that should and could be helpful is to add a way to sort your assignments or tasks by either date created or by the due date.
